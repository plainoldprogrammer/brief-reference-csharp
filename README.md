# Brief Reference C#
Brief reference of the C# programming language.

---

#### Use a namespace
```
using System;
```

#### Define a namespace
```
namespace CustomNamespace
{
}
```

#### Use a namespace
```
using CustomNamespace;
```

#### Define a class
```
public class SomeClass
{
}
```

#### Define a method
```
public void DoSomething()
{
}
```

#### Define a static method
```
public static void DoSomething()
{
}
```

#### Call to a method
```
DoSomething();
```
